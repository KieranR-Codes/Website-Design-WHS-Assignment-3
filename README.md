# Website-Design-WHS-Assignment-3
This is a textbook tutorial website for a fictional restaurant called 'Curbside Thai' designed to teach more of HTML's and CSS's features.

*Running this File:*
- Clone files as a ZIP archive
- Unzip the archive in a new folder
- Go to the "Curbside Thai" folder
- Run the "ct_start.html" file to launch the site
